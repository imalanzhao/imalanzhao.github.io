Arch Linux硬盘分区

三月 5, 2022 发布在 Linux

一直记不住硬盘分区的命令，恰巧最近又折腾Linux，特记录一下。

查看硬盘

使用fdisk -l命令查看本机所有的硬盘及已有的分区，从下面的输出可以看出要分区的硬盘是/dev/sda

root@archiso ~ # fdisk -l
Disk /dev/sda: 83.27 GiB, 89409191936 bytes, 174627328 sectors
Disk model: VBOX HARDDISK
Units: sectors of 1 * 512 = 512 bytes
Sector size (logical/physical): 512 bytes / 512 bytes
I/O size (minimum/optimal): 512 bytes / 512 bytes


Disk /dev/loop0: 653.41 MiB, 685150208 bytes, 1338184 sectors
Units: sectors of 1 * 512 = 512 bytes
Sector size (logical/physical): 512 bytes / 512 bytes
I/O size (minimum/optimal): 512 bytes / 512 bytes

查看是否是UEFI模式

使用ls /sys/firmware/efi/efivars命令查看是否处于UEFI模式，如果输出中没有报错，就说明处于UEFI模式。这个步骤与后面要进行的EFI分区有关。

进入分区工具

使用fdisk /dev/sda命令选择要建立分区的硬盘，并进入分区工具。

root@archiso ~ # fdisk /dev/sda

Welcome to fdisk (util-linux 2.37.4).
Changes will remain in memory only, until you decide to write them.
Be careful before using the write command.

Device does not contain a recognized partition table.
Created a new DOS disklabel with disk identifier 0xdf8be451.

Command (m for help):

如果 sda已经有了分区，要先删除分区(如果需要的话)。

创建GPT partition table

输入g创建一个新的GPT partition table

Command (m for help): g
Created a new GPT disklabel (GUID: AC6E21E4-3582-4E48-919A-028493CDD397).

创建EFI分区

输入n创建分区

Command (m for help): n
Partition number (1-128, default 1):
First sector (2048-174627294, default 2048):
Last sector, +/-sectors or +/-size{K,M,G,T,P} (2048-174627294, default 174627294): +512M

Created a new partition 1 of type 'Linux filesystem' and of size 512 MiB.

输入t改变分区类型，然后输入 1选择 ‘EFI System’

Command (m for help): t
Selected partition 1
Partition type or alias (type L to list all): 1
Changed type of partition 'Linux filesystem' to 'EFI System'.

创建Root分区

Command (m for help): n
Partition number (2-128, default 2):
First sector (1050624-174627294, default 1050624):
Last sector, +/-sectors or +/-size{K,M,G,T,P} (1050624-174627294, default 174627294):

Created a new partition 2 of type 'Linux filesystem' and of size 82.8 GiB.

创建文件系统

EFI分区要格式化成FAT文件系统，/dev/sda2格式化成ext4。

root@archiso ~ # mkfs.fat -F32 /dev/sda1
mkfs.fat 4.2 (2021-01-31)
root@archiso ~ # mkfs.ext4 /dev/sda2
